# @tronbrowser/browser-core

Browser abstraction over the Chromium fork (tabs, profiles, bookmarks, history)

> Status: **stub** — interfaces defined, implementation pending. Part of milestone **M0 (Monorepo)**.

## Install

```bash
pnpm add @tronbrowser/browser-core
```

## Scripts

- `pnpm build` — compile TypeScript to `dist/`
- `pnpm typecheck` — type-check without emitting
- `pnpm test` — run unit tests (vitest)

See the [PRD](../../docs/tronbrowser-prd.md) for the overall architecture.
